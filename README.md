# about_cat
국비지원 그린컴퓨터 백엔드 과정 1차 웹애플리케이션 팀프로젝트 

<조원>
김진성 : 상품, 깃허브
이후현 : 회원가입
안희영 : 상품, 관리자 상품
임슬범 : 관리자 회원, 장바구니

 <운영체제>
Window 10


 <통합개발환경 IDE>
STS 3.9.9


 <JDK버전>
JDK 11


 <DataBase>
Oracle DB


 <관리 Tool>
Maven


-------------------------------------------------------------------------------------

 <프로젝트 기술 스택>


Frontend : HTML, CSS, JavaScript, AJAX, JSP

Backend : Spring

Database : Oracle Database


-------------------------------------------------------------------------------------

 <프로젝트 구현 기능>

회원 (Member)
회원가입 / 로그인, 로그아웃 / 회원 탈퇴

상품 (Item)
상품 등록 / 상품 수정 / 상품 삭제 / 상품 리스트 (메인페이지, 둘러보기 페이지) / 상품 상세 페이지 / 상품 정렬

주문 (Order)   <예정>
상품 주문 / 주문 내역 조회 / 판매 목록(Admin) / 입금 확인처리(Admin) / 주문취소

장바구니 (Cart) <예정>
장바구니 담기 / 장바구니 조회 / 장바구니 삭제
